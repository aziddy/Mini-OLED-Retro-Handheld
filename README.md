## Component List

* Some PCB Power Switch
* Adafruit Feather 328P
* 4 Tactile Switches
* Double Sided Protoboard
* Some random LiPo battery I ripped from some portable speaker
* OLED Display 128x64 Pixel I2C, 0.96 inch, SSD1306 
* 3D printed parts
